## Long Description (shared — adapt formatting per store)

**Plume AI: Write Better, Faster**

Your personal writing assistant powered by your choice of AI provider. Check grammar, rewrite for impact, translate to any language—all without leaving your browser.

**Key Features**

- Grammar checking with inline error highlights and one-click "Fix all"
- AI rewrite (Polish) for full text or selection
- Tone presets: Shorter, Formal, Direct, Technical
- Translate to any supported language
- Manual mode: check text on demand instead of auto-detecting
- Built-in PII anonymization (emails, phone numbers) before AI calls
- Per-site permissions: grant access site-by-site in the popup
- Disabled domains blocklist to skip certain sites
- Keyboard shortcut: Ctrl+Shift+G (or MacCtrl+Shift+G on Mac)
- Model selection: choose which model each AI provider uses
- Test connection button to validate your API key

**Supported AI Providers**

- Gemini (Google)—free tier available
- Claude (Anthropic)
- ChatGPT (OpenAI)
- Mistral
- Ollama (local/self-hosted)

**Privacy First**

You bring your own API key. No data collection. Your API keys are stored locally on your device and never leave except in direct requests to your chosen provider. We don't proxy, log, or store anything. All features work entirely offline on your device.

**How It Works**

1. Install the extension and open the popup
2. Choose an AI provider and enter your API key (required)
3. Focus any text field on any website and click the ✦ button (or press Ctrl+Shift+G)
4. Choose your action: check grammar, rewrite, or translate
5. Review and accept the AI's suggestions with one click

**Open Source**

Plume AI is open source. Inspect the code on GitHub to verify exactly what it does: [https://github.com/tsouef/plume-ai](https://github.com/tsouef/plume-ai)

**Languages Supported**

English, British English, French, German, Spanish, Dutch, Portuguese (Portugal), Portuguese (Brazil), Chinese, Arabic, Farsi, Japanese
